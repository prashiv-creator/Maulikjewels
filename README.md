# Maulik Jewels – E-Commerce Website

Maulik Jewels is a premium jewelry e-commerce platform featuring curated collections of rings, necklaces, earrings, and bracelets. 

## ✨ Features
- **Modern UI**: Dark emerald and gold theme for a luxurious feel.
- **Product Catalog**: Advanced filtering by category and material, with multiple sorting options.
- **Shopping Cart**: Fully functional cart with `localStorage` persistence.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Admin Panel**: Secure dashboard to manage product inventory (Add, Edit, Delete).

## 🛠️ Technical Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Icons**: Font Awesome 6.
- **Fonts**: Playfair Display (Headings) and Outfit (Body).
- **Persistence**: Browser `localStorage`.

## 📂 File Structure
- `index.html`: Homepage with featured arrivals.
- `products.html`: Main collection browsing page.
- `product-detail.html`: Individual product showcase.
- `about.html`: Heritage and values of Maulik Jewels.
- `contact.html`: Inquiry and contact form.
- `admin-login.html`: Secure access for authorized staff.
- `admin-dashboard.html`: Inventory management portal.
- `styles.css`: Global design system and aesthetics.
- `script.js`: Main user-facing functionality.
- `admin.js`: Administrative functionality and CRUD operations.
- `assets/images/`: Optimized jewelry and hero images.
